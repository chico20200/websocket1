# Chat en Tiempo Real

Aplicación de mensajería en tiempo real construida con Node.js, Express y Socket.IO.

## Tecnologías

- Node.js
- Express
- Socket.IO

## Funcionalidades

- Registro de usuario con nombre
- Mensajes globales (a todos los conectados)
- Mensajes privados entre usuarios
- Lista de usuarios conectados en tiempo real
- Notificaciones de sistema al conectarse y desconectarse

## Cómo ejecutar

1. Instalar dependencias:

```bash
   npm install
```

2. Iniciar el servidor:

```bash
   node index.js
```

3. Abrir en el navegador:

http://localhost:3000
